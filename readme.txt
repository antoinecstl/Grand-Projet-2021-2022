Instructions :
-------------

	- Dans votre répertoire de travail créez le répertoire de projet GestionScolaire.

	- Dans GestionScolaire créez un répertoire data et télécharger les fichiers csv dedans.

    - Télécharger 4 fichiers python dans votre répertoire de projet 

		lib_commun.py : permet la lecture et ecriture de fichier csv, à utiliser tel qu'il est, pas besoin de modification.

		lib_gestionEtudiants.py : module pour gestion de données de fichier etudiants.csv, à modifier et remplir les structures des fonctions.

		lib_gestionNotes.py : module pour gestion de données de fichier notes.csv, à modifier et remplir les structures des fonctions.
		
		gestion_scolaire.py : lancer ce fichier pour tester l'application


	- Exécutez gestion_scolaire.py pour tester votre projet

Votre répertoire de projet :

	GestionScolaire

	|   lib_commun.py 

	|   lib_gestionEtudiants.py 

	|   lib_gestionNotes.py 

	|
	+---data 

	|       etudiants.csv 

	|       matieres.csv 

	|       notes.csv
